# Satang 프로젝트 개요

## 프로젝트 소개

**Satang**은 AI 기반의 지능형 콘텐츠 생성 및 협업 플랫폼입니다. 사용자가 다양한 소스(PDF, URL, 텍스트, 이미지, 오디오, Google 문서 등)를 업로드하고, AI와 대화하며, 자동으로 인포그래픽, 슬라이드, 마인드맵, 플래시카드, 퀴즈, 보고서를 생성할 수 있습니다.

### 핵심 가치 제안

- **통합 소스 관리**: 다양한 형식의 학습 자료를 하나의 플랫폼에서 통합 관리
- **AI 기반 콘텐츠 생성**: Gemini AI를 활용한 고품질 학습 콘텐츠 자동 생성
- **실시간 협업**: 노트북 공유 및 실시간 채팅 기능
- **개인화된 학습**: 사용자 정의 테마와 맞춤형 콘텐츠 생성

---

## 타겟 사용자

### 1. 학생 및 교육자
- 강의 자료 정리 및 요약
- 학습 플래시카드 및 퀴즈 생성
- 시험 준비를 위한 마인드맵 활용
- 협업 프로젝트를 위한 노트북 공유

### 2. 직장인 및 리서처
- 문서 자동 요약 및 보고서 생성
- 프레젠테이션 슬라이드 자동 생성
- 인포그래픽으로 데이터 시각화
- 팀 문서 공유 및 협업

### 3. 콘텐츠 크리에이터
- 아이디어 구조화를 위한 마인드맵
- 소셜 미디어용 인포그래픽 생성
- 교육 콘텐츠 대량 생산
- 템플릿 기반 디자인 재사용

## 핵심 기능

### 1. 노트북 관리 (Notebook Management)

노트북은 콘텐츠를 조직화하는 기본 단위입니다.

**기능:**
- 노트북 생성, 검색, 공유, 삭제
- 이모지로 시각적 구분
- 제목과 설명으로 콘텐츠 명시
- 공유 링크를 통한 협업

**사용 예시:**
```
1. "머신러닝 기초" 노트북 생성
2. 관련 PDF와 URL 소스 업로드
3. AI와 대화하며 콘텐츠 이해
4. 스투디오에서 학습 자료 생성
5. 팀원들과 노트북 공유
```

---

### 2. 소스 관리 (Source Management)

다양한 형식의 학습 자료를 업로드하고 처리합니다.

**지원 형식:**
- **PDF**: 텍스트 추출 및 요약
- **URL**: 웹페이지 스크래핑 및 분석
- **텍스트**: 직접 입력 또는 붙여넣기
- **이미지**: OCR 및 Gemini Pro 이미지 분석
- **오디오**: 전사 및 요약 (예정)
- **Google 문서/슬라이드/시트**: 직접 연동 (예정)

**처리 프로세스:**
```
업로드 → 텍스트 추출 → 요약 생성 → 검색 인덱싱
```

---

### 3. AI 채팅 (AI Chat)

Gemini 3 Flash API를 활용한 스트리밍 채팅 기능입니다.

**기능:**
- 실시간 스트리밍 응답
- 소스 인용 (Citations)
- 대화 기록 관리
- 멀티턴 대화 컨텍스트

**사용 시나리오:**
```
사용자: "이 PDF의 핵심 개념 3가지를 요약해줘"
AI: [스트리밍 응답 + 소스 인용]
사용자: "두 번째 개념을 더 자세히 설명해줘"
AI: [이전 대화 맥락 유지 + 추가 설명]
```

---

### 4. 스튜디오 콘텐츠 생성 (Studio Content Generation)

AI를 활용한 다양한 학습 콘텐츠 자동 생성입니다.

#### 4.1 인포그래픽 (Infographic)
- 시각적으로 매력적인 정보 그래픽 생성
- 사용자 정의 테마 적용
- PNG/PDF로 내보내기

#### 4.2 슬라이드 (Slides)
- 프레젠테이션 슬라이드 자동 생성
- Google Slides 직접 연동 (예정)
- PDF/PPTX로 내보내기

#### 4.3 마인드맵 (Mindmap)
- 계층적 아이디어 구조화
- 노드 간 관계 시각화
- 확대/축소 및 탐색

#### 4.4 플래시카드 (Flashcard)
- 암기 학습용 카드 생성
- 정답/오답 피드백
- 학습 진도 추적

#### 4.5 퀴즈 (Quiz)
- 객관식 및 주관식 문제 생성
- 자동 채점 및 해설
- 난이도 조절

#### 4.6 보고서 (Report)
- 구조화된 보고서 자동 생성
- 목차, 서론, 본론, 결론
- 소스 인용 및 참고문헌

---

### 5. 테마 관리 (Theme Management)

사용자 정의 디자인 테마로 콘텐츠의 시각적 일관성을 유지합니다.

**테마 구성:**
- 색상 팔레트
- 폰트 스타일
- 레이아웃 템플릿
- 아이콘 세트

**사용 예시:**
```
1. "내 브랜드" 테마 생성
2. 브랜드 색상과 로고 설정
3. 모든 생성 콘텐츠에 자동 적용
4. 팀원들과 테마 공유
```

## 기술적 특징

- **클라이언트-서버 분리**: Next.js App Router의 서버/클라이언트 컴포넌트 아키텍처
- **실시간 스트리밍**: AI 응답의 스트리밍 전송으로 빠른 사용자 경험
- **보안**: Supabase RLS(Row Level Security)로 데이터 접근 제어
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원
- **타입 안전성**: TypeScript와 Zod로 엔드-투-엔드 타입 안전성 보장

## 성공 지표

- 사용자 참여도: DAU/MAU 비율 > 30%

- 시스템 안정성: 가동 시간 99.9% 이상
- AI 응답 속도: 평균 3초 이내
